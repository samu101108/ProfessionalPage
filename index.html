Habilidades

HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, q ue significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

CSS

Cascading Style Sheets (CSS) é uma linguagem de folhas de estilo utilizada para definir a apresentação de documentos escritos em uma linguagem de marcação, como HTML ou XML. O seu principal benefício é a separação entre o formato e o conteúdo de um documento.

PHP

PHP (um acrônimo recursivo para "PHP: Hypertext Preprocessor", originalmente Personal Home Page) é uma linguagem interpretada livre, usada originalmente apenas para o desenvolvimento de aplicações presentes e atuantes no lado do servidor, capazes de gerar conteúdo dinâmico na World Wide Web.

Java Script

JavaScript é uma linguagem de programação interpretada. Foi originalmente implementada como parte dos navegadores web para que scripts pudessem ser executados do lado do cliente e interagissem com o usuário sem a necessidade deste script passar pelo servidor, controlando o navegador, realizando comunicação assíncrona e alterando o conteúdo do documento exibido.

Python

Python é uma linguagem de programação de alto nível, interpretada, de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte. Foi lançada por Guido van Rossum em 1991. Atualmente possui um modelo de desenvolvimento comunitário, aberto e gerenciado pela organização sem fins lucrativos Python Software Foundation.

Linux

Linux é um termo comumente utilizado para se referir a sistemas operacionais que utilizam o kernel Linux. O núcleo Linux foi desenvolvido pelo programador finlandês Linus Torvalds, inspirado no sistema Minix.

Window Server

Windows Server é uma família de sistemas operacionais Microsoft Windows baseado na arquitetura NT direcionada para uso em servidores.

GitHub

GitHub é um Serviço de Web Hosting Compartilhado para projetos que usam o controle de versionamento Git. É escrito em Ruby on Rails pelos desenvolvedores da Logical Awesome Chris Wanstrath, PJ Hyett e Tom Preston - Wernder). O GitHub possui planos comerciais e gratuitos para projetos de código aberto.

Android

Android é um sistema operacional (SO) baseado no núcleo Linux e atualmente desenvolvido pela empresa de tecnologia Google. Com uma interface de usuário baseada na manipulação direta, o Android é projetado principalmente para dispositivos móveis com tela sensível ao toque como smartphones e tablets.

SQL

Structured Query Language, ou Linguagem de Consulta Estruturada ou SQL, é a linguagem de pesquisa declarativa padrão para banco de dados relacional (base de dados relacional).

Photoshop

Adobe Photoshop é um software caracterizado como editor de imagens bidimensionais do tipo raster desenvolvido pela Adobe Systems. É considerado o líder no mercado dos editores de imagem profissionais, assim como o programa de fato para edição profissional de imagens digitais e trabalhos de pré-impressão.

Shell Script

Shell script é uma linguagem de script usada em vários sistemas operativos (operacionais), com diferentes dialetos, dependendo do interpretador de comandos utilizado. Um exemplo de interpretador de comandos é o bash, usado na grande maioria das distribuições GNU/Linux.

O que eu sou?

Brasileiro, Pernambucano, Recifence!

Ao longo dos últimos 12 anos eu adquiri um bom conhecimento da área de TI. Desde o primeiro computador que chegou em casa, onde eu bangunçava tudo, até agora onde eu sigo estrutura de dados e protocolos, posso dizer que sou um profissional sério, focado em resolver problemas e realizar o trabalho da melhor maneira possível.

Windows ou Linux não é o problema. A questão é: qual deles resolve de melhor forma aquele problema específico?

Eu tenho experiência em sistemas Windows, como o Microsoft Windows Server 2012 - configuração e implantação, assim como o último Microsoft Windows 10.

Não feliz o suficiente com sistemas corporativos "Dementadores", eu me apaixonei pelo Linux desde 2006 e suas linhas comandos são uma boa para mim. Shell Script é ótimo!

Tenho habilidades de Front-end com HTML5, CSS3, Java Script e Bootstrap Twitter, mas não parei por aqui e também sei um pouco de Python e Java.

E talvez por último, eu sou um singelo desenvolvedor Android também.

Estou procurando desafios em minha carreira. Eu sei que a ciência da computação e a engenharia são fascinantes e mudam vidas. Eu tenho o maravilhoso sentimento que estou fazendo parte de um todo em que através da tecnologia o mundo é transformado em um lugar melhor.
